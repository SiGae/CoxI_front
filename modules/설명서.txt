폴더별 간략 설명
Module
  데이터를 저장하는 곳을 관리하는 곳
api
  서버와 통신을 위한 곳
saga
  Module에서 공통적으로 사용하는 기능을 모은 곳.

Module 작성 및 사용법
1. Action 정의
2. Dispatch 될 함수
    파라미터엔 View로 부터 받을 데이터를 정의하고 
    return 값은 리듀서나 미들웨어에서 받을 데이터를 보낸다.
3. 서버와 통신 또는 중간에 데이터 처리를 하기 위한 
   미들웨어에 필요한 함수 작성 (종류에 따라 작성 방법이 다름) 
   Saga를 기준으로 제너레이터 함수 작성

4. 제너레이터 함수를 관리하는 함수 작성 (미들 웨어)
5. 리듀서 작성

3, 4의 경우 경우에 따라 생략 가능.
